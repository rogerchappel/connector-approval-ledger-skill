# Contributing

Thanks for helping improve `connector-approval-ledger-skill`.

## Development

Use Node.js 20 or newer.

```bash
npm install
npm run release:check
```

## Pull Requests

- Keep changes focused and fixture-backed when behavior changes.
- Run `npm run release:check` before opening a PR.
- Preserve dry-run behavior and avoid adding live connector side effects.

## Issues

Please include the command, relevant fixture shape, and expected versus actual output.
